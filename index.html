Learn Morse

Tired of using normal latin script alphabets to type? How about something way more cooler like Morse Code

If you didn't know already, Morse Code is a way of communication which uses beeps.
How? There are two standard beep called dit represented by dots and da-ah represented by dashes.

Each letter has a unique combination of them. Replace that for normal letters and you can communicate using morse in normal plain English but which only cool people like you (and me) can understand.
